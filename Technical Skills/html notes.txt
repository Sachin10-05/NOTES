7) Types of Elements:
---------------------
	ðŸ”’ Inline element:-
		* An inline element does not start on a new line.
	       	* An inline element only takes up as much width as necessary.
		* Inline elements don't have top and bottom margins.
            	Ex- <em>, <button>, <span>, <br>, etc.
		
	ðŸ”’ Block level element
            	* A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
            	* A block-level element always takes up the full width available.
		* Block element have both top and bottom margins.
            	Ex- <div>, <p>, <h1>, etc.

20) DIV & SPAN Tag:
--------------------
	ðŸ”’ <div> 
		* It is used as container for other HTML elements.
		* It has no required attribute but style,class,and id are common. 
		* It is a block level element.
		* But its default block-level property can be changed using css "display" attribute.
		  <style>
		       div {
	 	          display: inline;
		       }
		  </style>

	ðŸ”’ <span> 
		* It is an inline container used to mark up a part of a text, or a part of a document.
		* It has no required attribute but style,class,and id are common. 
		* It is a inline element.
		* But its default inline property can be changed using css "display" attribute.
		  <style>
		       span{
	 	          display: block;
		       }
		  </style>
